Código Abierto vs. Software Libre

Características del Código Abierto:

  1. Acceso al código fuente
  2. Permite la modificación y redistribución
  3. Colaboración comunitaria
  4. Transparencia
  5. Licencias permisivas

Características del Software Libre:

  1. Libertad para usar el software con cualquier propósito
  2. Libertad para estudiar cómo funciona el programa
  3. Libertad para distribuir copias
  4. Libertad para mejorar el programa y distribuir las mejoras
  5. Énfasis en la ética y los derechos de los usuarios

Publicado AlfCMdev 04 de Diciembre del 2024.

Características del Software Privado o Cerrado

  • Código fuente no disponible al público
  • Licencias restrictivas
  • Prohibición de modificación y redistribución
  • Desarrollo centralizado
  • Soporte técnico oficial

Arch Linux

Arch Linux es una distribución de Linux conocida por su enfoque de "hazlo tú mismo", lo que significa que proporciona un sistema base y permite a los usuarios personalizar completamente su entorno de software. Utiliza un modelo de actualización continua, lo que significa que no se realizan lanzamientos grandes, sino que el sistema se mantiene actualizado con las últimas versiones de paquetes. Arch es popular entre usuarios avanzados debido a su filosofía de simplicidad y control.

Personalmente, considero a Arch Linux como una de las distribuciones mas bonitas que existen, y es de hecho a la que me cambiare en estos futuros dias. Sin embargo, debo aprender bien a realizar la instalacion y todo lo que esto conlleva, ya que de entrada es solo una terminal, un lienzo en blanco para dejar volar la creatividad.

Esta es una imagen de un Escritorio de alguien de la comunidad, existen muchos mas y diferentes.
